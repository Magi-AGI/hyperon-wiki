12. S4.X.Q3 - Cluster-narrative placement
=========================================

Verdict:
  [IL-TRACK-FORMAL-SEMANTICS], with FormalMeTTa as [SPEC-REFERENCE-IMPLEMENTATION]
  only in the descriptive sense, not as the selected S4.X.Q3 label.

Rationale:
  MeTTaIL is best placed as the formal-semantics / intermediate-language track
  inside the MeTTa runtime cluster. It is neither a direct runtime peer of
  hyperon-experimental/PeTTa/MeTTaTron nor a pure paper artifact. It supplies
  GSLT grammar/interpreter/transforms and side prototypes for stochastic and
  matrix interpretations.

  FormalMeTTa is a compact archival reference/demo for an older preliminary
  MeTTa spec. It is useful for historical/spec comparison, but it should not be
  treated as an active runtime implementation line.
