```
[Fintype ι] (prior : BinaryEvidence) (likelihood : ι → BinaryEvidence) :
  BinaryEvidence.toStrength (nbEvidence prior likelihood) =
    nbPosterior prior.pos prior.neg
      (fun i => (likelihood i).pos) (fun i => (likelihood i).neg)
```
Proof body: `classical`, `unfold`, `simp` — full proof, no `sorry`.

**`PLN_hplusPos_eq_knnRelevance`** at `Mettapedia/Logic/PremiseSelectionKNN_PLNBridge.lean:125`:

```
{Fact : Type*} [DecidableEq Fact]
  (goal : Fact) (N : Finset Fact) (near : Fact → Fact → ℝ≥0∞)
  (deps : DepSet Fact) (tau2 : ℝ≥0∞) (phi : Fact) :
  (plnKnnEvidence goal N near deps tau2 phi).pos =
    knnRelevanceENN goal N near deps tau2 phi
```
Proof at line 130 delegates to `plnKnnEvidence_pos_eq_knnRelevanceENN`.
