**[Claude orchestrator, repo-verified]** Three orchestrator-side checks against canonical `3b356c5`:

**Check 1: eq (11) resolution_level absence in active modulator updater.**

```
$ git -C hyperon-openpsi grep -n "resolution\|sqrt" -- "main/mind-agents/modulator-updater/modulator-updaters.metta"
[NO MATCHES]
```

Verified ABSENT. The active modulator updater computes activation, securing-threshold, and selection-threshold but does NOT recompute resolution-level from activation. Paper eq (11) `resolution_level = 1 - sqrt(activation)` is unimplemented in the active updater.

**Check 2: eq (5) certainty demand denominator.**

`git show HEAD:main/mind-agents/demand-updater/demand-updaters.metta` lines 40-55 verbatim:

```metta
; Certainity demand updater
(: CertaintyDemandUpdater (-> Expression Number  Number))
(= (CertaintyDemandUpdater $lst  $ts)

    (let* (
            ($summation (sum-fuzzy-new $lst $ts ))
            ($object-num (size-atom $lst))
            ($rand (rand-float))
            ($x (exp (* -0.05  $object-num)))
        )
    (/ (+ $summation $rand)
        (+ 1 (* $x (+ 1  $object-num)))))
)
```

Code denominator: `(+ 1 (* $x (+ 1 $object-num)))` = `1 + exp(-0.05*n) * (1+n)`.
Paper denominator (per Codex C3-20 paper text): `(1 + exp(-0.05*n)) * (n+1)`.

These are NOT algebraically equal:
- Paper:  `(1 + e) * (1+n)` = `1+n + e + en`
- Code:   `1 + e*(1+n)` = `1 + e + en`

Difference: paper has `+n` term that code lacks. **DIVERGENT confirmed.**

**Check 3: eq (8) affiliation demand denominator.**

`git show HEAD:main/mind-agents/demand-updater/demand-updaters.metta` lines 57-69 verbatim:

```metta
; Affiliation Demand Updater
(: AffiliationDemandUpdater (-> Expression  Number Number))
(= (AffiliationDemandUpdater  $lst  $d-max)
    (let* (
            ($summation (sum-fuzzy-near $lst $d-max ))
            ($friend-num (size-atom $lst))
            ($rand (rand-float))
            ($x (exp (* -0.1  $friend-num)))
        )
    (/ (+ $summation $rand)
        (+ 1 (* $x (+ 1  $friend-num))))
))
```

Same parenthesization as eq (5). **DIVERGENT confirmed.**
