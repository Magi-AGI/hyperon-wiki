P7. Categorial state model (paper-1 §2):
    - Motivational state X = G × M in category C.
    - G = (g₁, ..., g_P) — vector of goal intensities.
    - M = (μ₁, ..., μ_K) — vector of modulator parameter values
      (valence, arousal, ethical vigilance, social drive, energy level,
      safety margin, etc.).
    - Comonad (Ψ, ε, δ) for appraisal: updates affect/modulators via
      Ψ(X) → X.
    - Monad (D, η, μ) for decision: scores/selects goals via X → D(X).
    - Lax distributive law λ_X : Ψ(D(X)) ⇒ D(Ψ(X)) — "feel then choose"
      vs "choose then feel" differ slightly but in controlled way.
    - F = D ∘ Ψ inherits both structures (pseudo-bimonad).
    - Motivational coalgebra α : X → F(X) governs one full cycle.

P8. Five design principles — VERIFIED IN PAPER-1, ALL FIVE PRESENT.
    Paper-1 abstract enumeration:
    (1) Modular Appraisal-Decision Interface (lax distributive law
        making "feel-then-choose" ≈ "choose-then-feel"). §3.1.
    (2) Reciprocal State Simulation (functor T mapping A's state to B's,
        natural transformation φ_X). §3.2.
    (3) Parallel Motivational Compositionality (lax-monoidal structure
        F(X₁) ⊗ F(X₂) → F(X₁ ⊗ X₂)). §3.3.
    (4) Homeostatic Drive Stability (boundary-driven contractivity:
        d(F(x), F(y)) ≤ c·d(x,y) + ϵ on boundary band B_η of region R).
        §4.1.
    (5) Incremental Objective Embodiment (gradual update F_α(x) =
        (1-α)x + α σ(τ(x))). §4.2 + §5.
    Note paper-1 introduction enumeration uses different ordering (1
    Modular A-D Interface; 2 Homeostatic Stability; 3 Reciprocal State
    Simulation; 4 Motivational Compositionality; 5 Incremental Objective
    Embodiment) — abstract list is canonical.
    Resolves S1.V1-2 concern: principles ARE paper-text from Lian &
    Goertzel 2025, NOT only a Goertzel 2025-09 SubRep paper reframing.
    The Goertzel 2025-09 SubRep §4 attribution is FAITHFUL to paper-1
