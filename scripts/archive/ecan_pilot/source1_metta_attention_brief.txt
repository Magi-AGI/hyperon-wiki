## Comparison targets (carryover from PLN cluster Source 11)

The PLN cluster pilot established that production PLN/ has no inference-control layer and that ECAN was the documented historical control aspiration. This Source 1 must answer:

1. **Are STI/LTI conservation laws enforced in MeTTa code, or just notional?** Look for: rent + wages balance, central-bank inflation/deflation, total-mass conservation invariants.
2. **Is the Hebbian Conjunction equation realized?** From the paper: `c_ij' = ConjDecay * (si * sj / recentMaxSTI^2) + (1 - (si * sj / recentMaxSTI^2)) * c_ij`. Verify literally in code.
3. **Is rent-extraction the paper formula?** From paper: `rent = Rent * max(0, log2(si / recentMaxSTI))^2`.
4. **Is diffusion stochastic per the paper?** From paper: `P(diffuse) = 0.5 * (1 + tanh(shape * (si - FocusBoundary)))`.
5. **Is there ECAN-PLN integration?** Cross-reference: does any code in `experiments/` or elsewhere couple this attention runtime to a PLN inference engine? If so, with which PLN repo (PLN/, PLN3, CeTTa lib_pln*)? See **integration-audit grep targets** below.
6. **Is AttentionalFocus a real data structure or a placeholder?** `attention-bank/bank/attentional-focus/attentional-focus.metta` — what does it actually do?
7. **Are the AF (Attentional Focus) vs WA (Whole AtomSpace) variants for diffusion/rent both implemented, or is one a stub?** Both directories exist; do both have non-trivial bodies?
