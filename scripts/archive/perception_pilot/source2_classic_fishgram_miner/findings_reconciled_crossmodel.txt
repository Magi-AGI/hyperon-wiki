maintenance), broke quietly when its dependency tree was reorganized

2020. The toy pipeline never reached robot-vision scale. The
"abandoned external-miners" comment in adaptors.py:30-33 hints that

binary integration in favor of an in-house Python re-implementation,

the algorithm-family literature (Chi 2005 survey paper) without any

zero involvement from the 2012-2013 FISHGRAM Python authors. ngeiswei/
miner is a strict-ancestor fork (8 commits behind, 0 ahead).
